# Assets — screenshots to add before publishing

Take these yourself (the dashboards contain real company data — redact before committing):

1. **top_targets.png** — the ⭐ Top Targets tab. Blur/crop the company-name, founder, and
   LinkedIn columns AND the 🟡 ACCOUNT / 🟢 COMMENTS columns entirely. The structure (score,
   Score Δ, Why Δ, momentum, stage columns) is what the reader needs to see, not the leads.
2. **daily_digest.png** — one day of the digest tab, same redaction rules.
3. **pipeline_status.png** — terminal screenshot of `pipeline_status.txt` (safe as-is, it's
   only stage names + OK marks).
4. Optional: **dashboard.png** — the per-country Dashboard tab with aggregate counts only
   (aggregates are safe; rows are not).

macOS: Cmd+Shift+4 to capture, then Preview → Markup → rectangle tool with solid fill for
redaction (do NOT use blur/pixelation tools for sensitive data — pixelation can be reversed;
solid fill cannot).
